class verry.linalg.FloatIntervalMatrix(inf, sup=None, **kwargs)#

Bases: IntervalMatrix[float]

Double-precision inf-sup type interval matrix.

alias of FloatInterval

transpose

property ndim#

Shorthand for len(self.shape).

Unlike NumPy’s ndarray, x.ndim is always greater than or equal to 1, i.e., there is no 0D matrix.

property shape#

Tuple of matrix dimensions.

Examples

>>> from verry.linalg import FloatIntervalMatrix as FIM
>>> x = FIM([[1, 2, 3], [4, 5, 6]])
>>> y = FIM([1.0, 2.0])
>>> x.shape
(2, 3)
>>> y.shape
(2,)
